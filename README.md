# To-do-app
A simple CLI To-Do application written in C for managing daily tasks.a learning project too!

---

### Features

- Add Task
- List Task 
- Delete tasks
- Formatted Output
- Uses functions for faster data handling

---

### Concepts that i got my hands on 

- `fgets()` 
- `switch()` statement
- Using `strcpy` for shifting arrays
- Delete Concept with arrays
- array handling with pointers

---

### Future improvements 

- more decisions under add, del, and list tasks
- task reminder, deadline Features
- aesthetic looking dashboard (idk why i am gonna make it this good!!)

---

### How to Compile and Run

**With gcc:**
```bash
gcc to-do.c -o todo && ./todo
```

**With Clang:**
```bash
clang to-do.c -o todo && ./todo
```
